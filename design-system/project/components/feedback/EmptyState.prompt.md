Centered empty state — outlined icon in a paper disc, serif headline, body copy, optional action.

```jsx
<EmptyState icon="shopping-cart" title="Your list is empty"
  action={<Button icon="plus">Add the first item</Button>}>
  Add items as you think of them, or import a meal plan from RecipeTracker.
</EmptyState>

<EmptyState icon="check" title="All checked off!">
  Nice work. Tap End trip to clear the list for next time.
</EmptyState>
```

Use for the fresh list, a fully-shopped store, or an empty import preview.
