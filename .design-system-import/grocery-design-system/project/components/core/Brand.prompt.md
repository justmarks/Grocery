The Grocery brand lockup — "Marks Family / Grocery" wordmark beside a tomato grocery-list monogram.

```jsx
<Brand />                       {/* horizontal lockup — top bar */}
<Brand variant="stacked" />     {/* sign-in / launch card */}
<Brand variant="mark" size={40} /> {/* monogram only */}
<Monogram size={28} />          {/* the glyph by itself */}
```

The monogram is drawn from CSS tokens, so it follows the palette. "Grocery" is set in Newsreader italic tomato, paralleling the Recipe Book's "Recipe Book" line.
